## 说明
批量任务执行框架，执行批量相同任务。

## 思路

线程池执行任务

成员
    
    任务池
       - 批量任务列表 
                 - 任务 
                       -   任务名
                           List<任务实体VO>
                           单任务执行抽象
                           成功执行列表
                           失败执行列表
                           任务执行失败次数限制 -> 失败策略实现？
                           是否延时任务 - 延时任务则 每批任务执行后，延时后执行第二次
       - 执行线程池
       - 任务检查类
       
            
       
           
    任务实体VO
    
    单任务执行抽象
        - 任务执行回调
    
    任务检查类 
        提供对任务的查询操作，任务回收。
        

## 支持springboot

## 使用batchJob任务


注解启用
@EnableBatchJob

继承BatchJobConfigurer 实现 Jobpool 对象初始化到spring的容器中！
